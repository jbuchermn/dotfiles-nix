# Setup NixOS

Follow NixOS manual, be sure to setup
- Networking / Wireless
- Hostname
- User and password
- `nix flake`
- `vim`
- `git`
on first setup.

Then clone this repo, (possibly add entries for user and hostname), add `hardware-configuration.nix` and apply.
